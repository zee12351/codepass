# Zarivo — Medical Coding Exam Prep

Practice questions, timed mock exams, and rationales for medical-coding certification exams (CPC, CCS, CCA) — English & Hindi.

## Structure
- `index.html` — marketing landing page
- `app.html` — student portal / diagnostic quiz
- `questions.js` — question bank
- `app.js` — quiz logic & utilities
- `styles.css` — design system

## Deploy
Static site, hosted on Vercel. Production domain: zarivo.in

```bash
vercel --prod
```
